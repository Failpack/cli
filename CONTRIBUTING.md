# Contributing

Thanks for your interest in FailPack.

This repository is currently used for public documentation, issues and release information.

The core product is under active early development. Public contribution guidelines will be expanded before the first stable release.

## Reporting bugs

Please open a bug report using the issue template.

## Suggesting features

Please open a feature request using the issue template.

## Security issues

Do not report security issues publicly. See `SECURITY.md`.
