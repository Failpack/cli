# FailPack CLI Documentation

FailPack CLI is installed globally from npm:

```bash
npm install -g @failpack/cli
```

Run:

```bash
failpack
```

## Planned CLI commands

```bash
failpack
failpack capture
failpack ai
failpack report
failpack doctor
failpack clean
failpack init
failpack login
failpack logout
failpack whoami
failpack status
failpack upgrade
failpack cloud upload
failpack analyze
```

## Local-first behavior

FailPack is designed to generate local debug reports by default. Cloud features will require explicit authentication and user confirmation.
